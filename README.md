## 安装

  Using IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "intellij-plugin-yapi2ts"</kbd> >
  <kbd>Install Plugin</kbd>
